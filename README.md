Financial Sample Dashboard (Power BI)
Overview

This project is an interactive Financial Performance Dashboard built using Power BI, designed to analyze and visualize key business metrics from a sample financial dataset. It helps track revenue, profit, and sales trends across regions, products, and customer segments.

Objective

The goal of this dashboard is to provide a clear and dynamic view of financial performance to support data-driven decision-making.

Key Insights

Total revenue, profit, and sales growth over time

Regional and country-level performance comparison

Top-performing products and customer segments

Profit margin analysis by product and region

Dynamic filters for year, segment, and country

Tools Used

Power BI – For data modeling, visualization, and dashboard creation

Microsoft Excel – For storing and cleaning the financial dataset

Files in Repository

Financial Sample.xlsx – Raw dataset used for analysis

Financial Sample Dashboard.pbix – Power BI project file

README.md – Project documentation

How to Use

Download both files.

Open Financial Sample Dashboard.pbix in Power BI Desktop.

Review visuals and apply filters to explore insights interactively.
